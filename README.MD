# Rust demo

Minimal rust project

```sh
# Build dev/debug
cargo build
#Build release
cargo build --release

# create helloworld
cargo new test_rust

# run
cargo run

# Run:
$PROJECT_ROOT/target/release/rust-demo1

# inside debug shell
print arr
#shows: (int[3])  ([0] = 1, [1] = 2, [2] = 3)

```
